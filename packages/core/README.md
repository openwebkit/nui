# @openwebkit/nui

A small UI component package built with Tailwind and Vite.

Install

```bash
npm install @openwebkit/nui
```

Usage

```tsx
import React from "react";
import { Button } from "@openwebkit/nui";

export default function App() {
  return <Button>Click me</Button>;
}
```

See the repository for examples and Storybook.
